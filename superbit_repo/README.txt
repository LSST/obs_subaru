Real SuperBIT data from 2017pre-flight

Calib data: 
As we only have Dark image, flat and bias images are using the same dark images as an input to test the obs_SuperBIT.

registry.sqlite3 is generated by running:  
superbitIngestImages.py /superbit_repo/ DATA_FILES/raw/*.fits

calibRegistry.sqlite3 is generated by 
ingestCalibs.py superbit_repo/ --calib superbit_repo/CALIB 'DATA_FILES/CALIB/DARK/dark*.fits' --validity 99 
ingestCalibs.py superbit_repo/ --calib superbit_repo/CALIB 'DATA_FILES/CALIB/FLAT/flat*.fits' --validity 99 
ingestCalibs.py superbit_repo/ --calib superbit_repo/CALIB 'DATA_FILES/CALIB/BIAS/bias*.fits' --validity 99
